产品描述及出借说明

七天大胜是新新贷推出的[[${globalData.detailQTDS.data.period}]]天期、体验型出借计划，每日24小时开放出借，随时可投。

该产品[[${globalData.detailQTDS.data.lowestTender}]]元起投，出借上限[[${#numbers.formatDecimal(globalData.detailQTDS.data.mostTender,1,'COMMA',2,'POINT')}]]元， 每人限投一次。该产品一旦出借成功，不支持提前赎回。 该产品债权对接平台的优质短期借款标的。

收益如何计算

本产品出借成功后当日开始计息，新用户（即尚未在新新贷平台出借过的用户）出借历史年化收益率为[[${globalData.detailQTDS.data.apr + globalData.detailQTDS.data.floatApr}]]%（产品历史年化收益率[[${globalData.detailQTDS.data.apr}]]%+平台额外贴息[[${globalData.detailQTDS.data.floatApr}]]%），老用户出借历史年化收益率为[[${globalData.detailQTDS.data.apr}]]%。

收益=出借金额×[[${globalData.detailQTDS.data.apr + globalData.detailQTDS.data.floatApr}]]%(或[[${globalData.detailQTDS.data.apr}]]%)÷360×[[${globalData.detailQTDS.data.period}]]

1.我注册你们平台有段时间了，还没有出借过，我可以出借七天大胜吗？

当然可以，七天大胜是新老用户都可以出借的体验性出借计划，每人限投1次，新用户出借还能再享加息5%。

2.我之前出借过你们平台的其他产品，但是没有出借过七天大胜，我现在可以出借七天大胜吗？

可以的，为回馈老用户，七天大胜现在向老用户也开放体验，只要之前没有出借过七天大胜，每人都有一次体验机会，只限1次。

3.收益什么时候到账？

成功出借本产品之日起7天（含出借当日及节假日）后，例如您在11月1日出借，11月8日就可以收到本金加收益。

4.我出借了100元，还可以再投吗？

不可以了，七天大胜只能出借一次。

5.我11月1日出借了这个产品，11月4日想提前退出了，怎么退出？

您的本金加收益将会在11月8日自动退出，该产品一旦出借，不支持提前赎回。
